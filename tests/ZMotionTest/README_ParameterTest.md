# 参数测试功能使用说明

## 功能概述

参数测试功能提供了对ZMotion控制器的GetParam和SetParam方法的完整测试界面，允许用户方便地读取和设置轴参数。

## 功能特性

### 1. 单个参数操作
- **参数读取 (GetParam)**
  - 支持选择任意可读参数
  - 实时显示读取结果
  - 常用参数快速选择按钮
  - 详细的状态反馈

- **参数写入 (SetParam)**
  - 支持选择任意可写参数  
  - 数值输入验证
  - 安全提示和注意事项
  - 操作结果确认

### 2. 批量参数读取
- 一键读取所有常用参数
- 表格形式显示结果
- 支持单个参数刷新
- 读取状态和时间记录

## 支持的参数类型

### 可读参数 (ReadBaiscParmName)
包含85+个可读参数，主要包括：

**位置相关**
- `Dpos` - 轴指令规划位置
- `Mpos` - 编码器反馈位置
- `Endmove` - 当前运动目标位置
- `Encoder` - 编码器原始值

**速度相关**  
- `Speed` - 运行速度
- `Mspeed` - 实际编码反馈速度
- `Vp_Speed` - 当前运行速度
- `Lspeed` - 起始速度
- `Creep` - 接近速度

**加减速相关**
- `Accel` - 加速度
- `Decel` - 减速度  
- `Fastdec` - 快减减速度
- `Sramp` - 加减速曲线时间设置

**状态相关**
- `Idle` - 运动状态
- `Axisstate` - 轴状态
- `Axis_Stop_Reason` - 轴停止原因
- `Loaded` - 缓冲是否为空状态位

**配置相关**
- `Atype` - 轴类型
- `Units` - 脉冲当量
- `Fs_Limit` - 正向软限位设置
- `Rs_Limit` - 反向软限位设置

### 可写参数 (WriteBaiscParmName)  
包含60+个可写参数，主要包括上述参数中的可配置项。

## 使用步骤

### 基本使用
1. 启动应用程序
2. 点击左侧导航菜单中的"参数测试"
3. 确保设备已连接
4. 选择要操作的轴索引
5. 进行参数读取或写入操作

### 单个参数读取
1. 在"单个参数"标签页的左侧面板中
2. 输入轴索引（如：0）
3. 从下拉列表选择要读取的参数
4. 点击"读取参数"按钮
5. 查看读取结果

### 单个参数写入
1. 在"单个参数"标签页的右侧面板中
2. 输入轴索引（如：0）  
3. 从下拉列表选择要写入的参数
4. 输入要设置的数值
5. 点击"写入参数"按钮
6. 确认操作结果

### 批量参数读取
1. 切换到"批量读取"标签页
2. 输入轴索引（如：0）
3. 点击"批量读取"按钮
4. 在表格中查看所有常用参数的当前值
5. 可单击"刷新"按钮更新特定参数

## 注意事项

### 安全提示
- 确保设备已正确连接后再进行操作
- 写入参数前请确认参数值的正确性
- 某些参数可能需要特定的权限或状态  
- 不正确的参数值可能影响设备运行

### 操作建议
- 建议先使用读取功能了解当前参数状态
- 写入参数时建议先备份当前值
- 对于关键参数，建议小幅调整并测试效果
- 遇到错误时请查看状态栏的详细信息

## 错误处理

应用程序提供了完善的错误处理机制：
- 连接状态检查
- 参数有效性验证
- 详细的错误信息显示
- 操作结果确认

所有操作结果都会在状态栏中显示，包括：
- 操作成功/失败状态
- 具体错误原因
- 操作时间戳
